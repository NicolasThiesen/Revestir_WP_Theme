## Instructions

- Go to the theme directory
`/wp-content/themes`
- Clone the repository
```shell
git clone https://github.com/NicolasThiesen/Revestir_WP_Theme.git
```
- Start the gulp in the project url
```shell
gulp
```
- Now you can start editing the theme or make changes
